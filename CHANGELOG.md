# Changelog

Все заметные изменения проекта документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [1.0.0] - 2026-06-21

### Added
- **Базовый функционал:** Отслеживание статусов эксплойтов и актуальных версий Roblox.
- **Интеграция API:** Подключение к `weao.xyz` для получения данных в реальном времени.
- **Избранное:** Возможность добавлять эксплойты в избранное с локальным сохранением (`SharedPreferences`).
- **sUNC тесты:** Подробное отображение результатов sUNC-тестов с группировкой по категориям.
- **Офлайн-кеширование:** Сохранение последних успешных ответов API для работы без интернета.
- **Интерфейс:** Тёмная и светлая темы, навигация через нижнее меню (`go_router`).

## [1.1.0] - 2026-06-23

### Added
- **Локализация (l10n):** Добавлена полная поддержка русского и английского языков через `.arb` файлы и пакет `intl`.
- **Dynamic Color:** Внедрена поддержка системных цветов Material 3 (Monet) для Android 12+.
- **Иконка приложения:** Добавлена кастомная иконка приложения с помощью `flutter_launcher_icons`.
- **Расширенная фильтрация и сортировка:** Добавлены новые фильтры для списка эксплойтов (по платформам, статусу "только бесплатные") и улучшена логика сортировки.
- Релизная подпись Android через `android/key.properties` с откатом на debug-подпись.
- Конфигурация базового URL и User-Agent через `--dart-define`.
- Логирование ошибок кеша и сетевых ошибок (раньше проглатывались молча).
- Юнит-тесты data-слоя (`Exploit`, `SuncData`, `RobloxVersions`, `CachedResult`).
- Изолированные виджет-тесты (файловый кеш / карточка эксплойта / баннер устаревших данных).
- Тесты `WeaoRepository` с фейковым API-клиентом.
- `LICENSE` (MIT), `README.md` с описанием проекта, `CHANGELOG.md`.
- Design tokens (`AppDimens`) для spacing/radii.

### Changed
- **Riverpod 3.x & Генерация кода:** Пакеты `flutter_riverpod` и `riverpod_annotation` обновлены до последних версий. `StateNotifier` заменён на современный `Notifier` API. Добавлена кодогенерация провайдеров (`@riverpod`).
- **Freezed:** Обновлён до версии 3.2.5. Включена явная сериализация `explicitToJson: true` в `build.yaml`.
- **Go Router 17.x:** Обновление пакета маршрутизации с рефакторингом API.
- `.gitignore` был расширен для исключения кэша генераторов кода (`.dart_tool/`, `*.freezed.dart`, `*.g.dart`) и приватных ключей.
- `WeaoApiClient` возвращает типизированные `Map`/`List` вместо `dynamic`.
- Детальный экран эксплойта использует переданный через роутинг объект вместо повторного запроса.
- Списки в детальном экране переведены на slivers (`CustomScrollView`) для ленивого построения.
- `analysis_options.yaml` — включены строгие lint-правила (`avoid_dynamic_calls`, `require_trailing_commas` и др.).

### Fixed
- Исправлен баг парсинга сложных вложенных JSON-структур (блок `sunc`) для моделей `Exploit` и `SuncData`.
- `getExploit` теперь кешируется и обрабатывает ошибки (раньше падал без офлайн-фолбэка).
- Ошибки pull-to-refresh больше не показывают красный баннер (обёрнуты в try/catch).
- Race condition избранного — загрузка из `SharedPreferences` до `runApp`, без мерцания UI.
- `_parseInt` теперь парсит строковые числа (`"97"`), бейдж процента не пропадает.
- `CachedResult` инвариант: `cachedAt` непустой при `isStale: true` (assert).
- Убраны русские (хардкод) строки из UI и API слоёв — перенесены в `.arb` словари.
- Дублирующий switch platform icon/label свёрнут в одну функцию-источник.
- sUNC-тесты в раскрытых группах строятся лениво (`ListView.builder`), без лагов первого кадра при сотнях тестов.
- Магические числа UI вынесены в design tokens (`AppDimens`).

## [1.2.0] - 2026-06-23

### Added
- **Sentry.io Интеграция:** Добавлен автоматический мониторинг ошибок, сбор стектрейсов и аналитика производительности.
- **CI/CD пайплайн:** Настроен GitHub Actions (`flutter_ci.yml`) для прогона тестов, линтера и сборки релизного APK-файла.
- **Анимации UI:** Добавлен пакет `flutter_animate` для плавного появления карточек эксплойтов и красивой скелетон-загрузки (Shimmer).
- **Сетевая отказоустойчивость:** Добавлен `dio_smart_retry` — автоматические повторные запросы при обрывах соединения с экспоненциальной задержкой.

### Changed
- **Офлайн-кеширование на Hive CE:** Устаревший JSON-кеш на базе файловой системы и `shared_preferences` полностью заменен на сверхбыструю NoSQL БД `hive_ce`.
- **Запуск тестов:** Исправлены таймауты и race conditions в виджет-тестах при использовании `flutter_animate` и `Hive`.

### Fixed
- Устранены дублирующиеся логи при сетевых запросах.
